# local-ai-pixel-farm

ローカルAI育成ファームは、農作業の記録を「エサ」としてピクセルモンスターに与えながら育てる、React + Vite 製のローカル実行向け育成アプリです。

農作物・ほ場と紐づくモンスターを作成し、水やり・施肥・観察・収穫・防除などの作業カテゴリをもとに、モンスターのステータス、反応、成長段階、エサやり履歴を管理します。現在のAI部分はローカルで動くルールベース実装で、将来的に本物のローカルAIへ差し替えやすい構成になっています。

## 起動方法

### 1. 依存関係をインストール

```bash
npm install
```

### 2. 開発サーバーを起動

```bash
npm run dev
```

表示された Vite のローカルURL（通常は `http://localhost:5173/`）をブラウザで開いてください。

### 3. 本番ビルドを確認

```bash
npm run build
```

### 4. ビルド結果をプレビュー

```bash
npm run preview
```

## 開発コマンド

| コマンド | 内容 |
| --- | --- |
| `npm install` | 依存パッケージをインストールします。 |
| `npm run dev` | Vite の開発サーバーを起動します。 |
| `npm run build` | TypeScript のビルドチェック後、Vite で本番ビルドします。 |
| `npm run preview` | `dist/` のビルド結果をローカルで確認します。 |

## 現在の機能

- 初回起動時にモンスターの名前、作物名、ほ場名、作物カテゴリを入力してタマゴを作成できます。
- モンスターは Canvas 上のピクセルアートとして表示され、成長段階・機嫌・行動に応じて描画が変化します。
- ホーム画面から水やり、施肥、観察、収穫、防除、その他のクイックエサやりができます。
- エサやり時に「エサが出る → 半分食べる → 食べ終わる → リアクションする」という簡易アニメーションが動きます。
- エサカテゴリごとのルールにより、元気・機嫌・健康・成長度が変化します。
- 成長度に応じてモンスターの成長段階が更新され、段階が上がると進化メッセージが表示されます。
- 任意の名前・メモ付きでエサを作成し、エサ一覧に保存できます。
- クイックエサやりの履歴を記録画面で確認できます。
- アプリ状態は `localStorage` に保存され、ブラウザを再読み込みしても継続できます。

## 今後の予定

- エサ一覧で選択した作成済みエサを、ホーム画面の給餌シーケンスと同じ処理で実際に与えられるようにする。
- 作成済みエサを使った場合も履歴に残す。
- `LocalBrain` をルールベースから実際のローカルAI推論へ差し替える。
- モンスターの個体差、作物カテゴリ、季節、ほ場状態などを反応判定に反映する。
- サンプルエサや外部農業データの取り込み導線を整理する。
- リセット、編集、削除などの管理機能を追加する。
